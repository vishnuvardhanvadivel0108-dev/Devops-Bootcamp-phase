# Topic 17 - Creating a Repository

## Objective

Learn how to initialize a Git repository, clone an existing repository, and understand the repository structure.

### Commands Practiced

#### Initialize a Repository

```bash
git init
```

#### Check Repository Status

```bash
git status
```

#### Add a File

```bash
git add README.md
```

#### Commit Changes

```bash
git commit -m "Initial commit"
```

#### View Commit History

```bash
git log --oneline
```

#### Clone a Repository

```bash
git clone <repository-url>
```

#### View Repository Structure

```bash
find . -maxdepth 2
```

## Conclusion

Practiced creating a local Git repository, making the first commit, cloning a repository, and understanding the repository structure.
