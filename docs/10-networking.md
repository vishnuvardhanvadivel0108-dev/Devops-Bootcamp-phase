# Topic 10 - Networking

## Objective
Learn basic Linux networking commands.

### Commands Practiced

#### hostname
Displays the hostname.

Example:

```bash
hostname
```

#### ip addr
Displays IP addresses and network interfaces.

Example:

```bash
ip addr
```

#### ip route
Displays the routing table.

Example:

```bash
ip route
```

#### ping
Tests network connectivity.

Example:

```bash
ping -c 4 google.com
```

#### nslookup
Performs DNS lookup.

Example:

```bash
nslookup google.com
```

#### dig
Retrieves DNS information.

Example:

```bash
dig google.com
```

#### curl
Retrieves data from a URL.

Example:

```bash
curl https://www.google.com
```

#### wget
Downloads files from a URL.

Example:

```bash
wget https://example.com
```

#### ss -tuln
Displays listening TCP and UDP ports.

Example:

```bash
ss -tuln
```

#### ss -s
Displays socket statistics.

Example:

```bash
ss -s
```

#### ip link
Displays network interfaces.

Example:

```bash
ip link
```

## Conclusion

Practiced checking IP addresses, routing, DNS resolution, network connectivity, and listening ports using common Linux networking commands.
