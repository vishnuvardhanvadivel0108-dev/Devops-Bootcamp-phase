# Linux File System

## Commands Practiced

### pwd
Displays the current working directory.

### cd /
Changes to the root directory.

### ls
Lists files and directories.

### ls -l
Displays a detailed listing.

### ls -la
Displays all files, including hidden files.

### cd /home
Navigates to the home directory.

### cd /etc
Navigates to the configuration directory.

### cd /var
Navigates to the variable data directory.

### cd /tmp
Navigates to the temporary directory.

### cd /usr
Navigates to the user programs directory.
