1. What is Linux?

Linux is an open-source operating system kernel created by Linus Torvalds in 1991. It manages computer hardware and allows applications to run efficiently. Linux is widely used in servers, cloud computing, embedded systems, and DevOps environments.

2. Why Linux is Used in DevOps?

Linux is the preferred operating system in DevOps because:

- Most cloud servers run Linux.
- It provides a powerful command-line interface.
- It supports automation through shell scripting.
- It is secure and stable.
- It integrates well with DevOps tools like Docker, Kubernetes, Jenkins, Git, and Terraform.

3. Linux Distributions

 * Ubuntu
- Beginner-friendly
- Debian-based
- Widely used in development and cloud environments

* Amazon Linux
- Developed by AWS
- Optimized for AWS services
- Commonly used on EC2 instances

* RHEL (Red Hat Enterprise Linux)
- Enterprise-grade Linux distribution
- Commercial support available
- Widely used in large organizations

4. CLI vs GUI

*CLI
- Uses text commands
- Faster for administration 
- Low resource usage
- Preferred by DevOps engineers 
*GUI
- Uses graphical interface 
- Easier for beginners
- Higher resource usage
- Common for desktop users

5. What is a Shell?

A shell is a command-line interpreter that allows users to interact with the operating system by executing commands.

Common shells include:
- Bash (Bourne Again Shell)
- Zsh
- Sh
- Ksh

6. Opening and Using the Terminal

 Open the terminal
- Ubuntu: Ctrl + Alt + T
- Amazon Linux: Connect using SSH
- RHEL: Applications → Terminal
