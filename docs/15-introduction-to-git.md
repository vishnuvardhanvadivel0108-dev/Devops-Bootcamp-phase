# Topic 15 - Introduction to Git

## Objective

Understand the fundamentals of Git and version control.

### What is Version Control?

Version Control is a system that tracks changes made to files over time. It allows developers to collaborate, maintain history, and restore previous versions if needed.

### Why Git is Used

- Tracks code changes.
- Enables team collaboration.
- Maintains version history.
- Supports branching and merging.
- Integrates with GitHub and CI/CD tools.

### Git vs GitHub

| Git | GitHub |
|-----|---------|
| Version control tool | Cloud hosting platform |
| Works locally | Stores repositories online |
| Tracks file changes | Enables collaboration |

### Basic Git Commands

```bash
git --version
git config --global --list
git status
git log
git log --oneline
git remote -v
```

## Conclusion

Learned the fundamentals of Git, version control, GitHub, and basic Git commands.
