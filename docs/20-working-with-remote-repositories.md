# Topic 20 - Working with Remote Repositories

## Objective

Learn how to connect and synchronize a local Git repository with GitHub.

### Commands Practiced

#### View Remote Repositories

```bash
git remote -v
```

#### View Remote Details

```bash
git remote show origin
```

#### Push Changes

```bash
git push origin main
```

#### Fetch Changes

```bash
git fetch origin
```

#### Pull Changes

```bash
git pull origin main
```

#### View Branch Tracking

```bash
git branch -vv
```

#### Check Repository Status

```bash
git status
```

## Conclusion

Practiced viewing remote repositories, pushing commits, fetching updates, pulling the latest changes, and verifying synchronization between the local repository and GitHub.
