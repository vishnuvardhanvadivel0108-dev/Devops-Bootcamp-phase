# Topic 18 - Daily Git Commands

## Objective

Learn the basic Git commands used in daily development.

### Commands Practiced

#### Check Repository Status

```bash
git status
```

#### Stage Changes

```bash
git add git-practice.txt
```

#### Commit Changes

```bash
git commit -m "Added git-practice.txt"
```

#### View Commit History

```bash
git log --oneline
```

#### Compare Changes

```bash
git diff
```

#### Create a .gitignore File

```bash
touch .gitignore
```

Example:

```text
*.log
*.tmp
```

## Conclusion

Practiced checking repository status, staging changes, committing work, comparing file changes, viewing commit history, and using `.gitignore` to exclude unnecessary files.
