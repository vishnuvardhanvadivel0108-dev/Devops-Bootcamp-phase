# Topic 15 - Git & GitHub

## Objective

Learn basic Git operations for version control.

### Commands Practiced

#### Check Git Version

```bash
git --version
```

#### View Configuration

```bash
git config --global --list
```

#### Check Repository Status

```bash
git status
```

#### View Commit History

```bash
git log --oneline
```

#### Create a Branch

```bash
git branch feature-module2
```

#### Switch Branch

```bash
git checkout feature-module2
```

#### Commit Changes

```bash
git add .
git commit -m "Commit message"
```

#### Merge Branch

```bash
git checkout main
git merge feature-module2
```

#### Push to GitHub

```bash
git push origin main
```

## Conclusion

Practiced creating branches, committing changes, merging branches, and pushing updates to GitHub.
